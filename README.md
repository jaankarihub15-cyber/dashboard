# My Dashboard

Interactive project dashboard with editable data. Add, edit, and delete rows and columns directly from the UI. Data auto-saves to your browser's localStorage.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Click **Deploy** (no settings to change — it auto-detects Next.js)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- Add / edit / delete rows and columns
- Search and filter across all data
- Auto-updating charts (pie, bar)
- CSV import and export
- Data persists in browser localStorage
